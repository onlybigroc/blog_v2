注意：SYS用户下执行。sys登录必须为sysdba身份。  
查看数据文件存放位置。

select \* from dba\_data\_files;

1、创建表空间

CREATE TABLESPACE tp\_lp
DATAFILE 
'd:\\data\\tp\_lp01.dbf' SIZE 10M
AUTOEXTEND OFF;\--是否自动扩展

不建议使用自动扩展

2、查询表空间

SELECT file\_name,tablespace\_name,bytes,autoextensible 
FROM dba\_data\_files 
WHERE tablespace\_name\='tp\_lp';

3、创建用户

CREATE USER bigroc IDENTIFIED BY bigroc
DEFAULT TABLESPACE tp\_lp;
\--PASSWORD EXPIRE;

4、授权（角色）

GRANT connect, resource TO bigroc; \--授予CONNECT和RESOURCE两个角色

5、撤销权限

REVOKE CONNECT , RESOURCE FROM bigroc;\--撤销CONNECT和RESOURCE两个角色
