# 一、凡会议，必有准备

1.  会议最大的成本是时间，会议没有结果就是对公司和参会人的犯罪，没有准备的会议等于“集体谋杀”；
2.  会议组织者应避免无准备的会议，当会议关键人缺席，可取消会议或者采用其他会议连线形式；
3.  重要会议前，必须把会议材料（议题、议程等）提前发给参会人员，参会人员要提前阅读材料并做好准备；
4.  会议参与人可分为必选和可选，可选人员可以自行选择是否参与会议；
5.  会议类型包括评审、决策、培训、头脑风暴等。

# 二、凡会议，必有纪律

1.  主持人应在会议开始前宣布纪律并在会议中坚决执行。
2.  对于迟到、不按流程进行、发言人带情绪、开小会私下讨论、会上发恶劣脾气、攻击他人等不当行为坚决抵制。
3.  设定时间，准时开始、准时结束。主持人对每个议程制定大致的时间限制，单一议题不能讨论过久，如不能得出结论可暂放，避免影响其他议题。

# 三、凡会议，必有结果

1.  开会的提纲是解决问题。
2.  鼓励每个人都积极地参与到会议中来，会议组织者应及时打断那些偏离会议主题的发言。
3.  会议主持人要进行时间提醒，例如“现在还有10分钟”。
4.  会议的决议要形成文字记录，进行宣读确认，重要内容需要每人确认。

# 四、凡会议，必有记录

1.  会议纪要须包括结论、行动点和细节3个要素；行动点要有具体的负责人及完成期限。
2.  如此项决议的完成需要多方资源，一定要在决议记录中明确说明，避免会后相互推诿，影响此决议的完成。
3.  “散会不追踪，开会一场空”，公司鼓励会议事后进行追踪跟进，保证结果落地。

# 五、凡会议，牢记三个公式

1.  开会+不落实=零！
2.  布置工作+不检查=零！
3.  抓住不落实的事+追究不落实的人=落实！

# 六、注意事项

1.  公司支持员工根据工作需求，自主发起会议活动，比如产品需求沟通、项目评审、部门间沟通协同等；
2.  大家可通过企业微信“日程”中会议发起功能组织召开会议，要求线上信息填写准确、详细，关联相关参与人，可选人员可以在备注中说明；
3.  被关联参会人员应第一时间确认能否参加，可参加选“接受”，不可参加选“拒绝”，什么都不选择默认为接受该日程安排；
4.  会议发起人作为组织者，应按照上述要求，是会议管理的主要责任人，提前做准备，现场有管控，后续有落实；
5.  最后，凡是使用会议室的同事，请提前在企业微信“日程”上或者“会议室”中进行预约，严格按照预约时间执行。
